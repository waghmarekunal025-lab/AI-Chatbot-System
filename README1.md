# 💬 Real-Time Generative AI Chatbot System

A responsive, production-ready web application that mimics the foundational features of ChatGPT. This system establishes a secure data handshake layer with Google's advanced `gemini-2.5-flash` live engine model endpoints to process operational natural language inputs and deliver streaming, real-time code execution structural breakdowns.

Live Production URL: [Click Here to View Live App](https://ai-chatbot-system-lujucbtwp9vydfjnmvxkk8.streamlit.app/)

---

## 🚀 Key System Features

* **Secure API Access Lock:** Features an authentication layer in the sidebar to prevent credential leakage.
* **Persistent Session State Memory:** Retains ongoing contextual chat message history structures so users can ask contextual follow-up questions.
* **Advanced Prompt Processing:** Capable of generating operational programming block syntax structures, logical breakdowns, and algorithm analysis instantly.
* **Ultra-Lightweight Cloud Architecture:** Optimized utilizing the highly efficient `uv` dependency compiler array, dropping cloud deployment load speeds to under 5 seconds.

---

## 🛠️ Tech Stack & Engineering Base

* **Core UI Layout Layer:** Streamlit (Python Web Framework Architecture)
* **Inference Compute Engine:** Google GenAI SDK Client (`gemini-2.5-flash`)
* **Data Validation Layer:** Pydantic v2 Core Type Engine
* **Cloud Virtualization Hosting Container:** Streamlit Community Cloud Ecosystem
